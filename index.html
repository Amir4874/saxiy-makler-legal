Sahiy Makler

Real Estate Video Publisher for the Uzbekistan Market

About

Sahiy Makler is an AI-assisted tool that helps real estate professionals in Uzbekistan turn property listings into short, narrated videos and publish them to social media platforms — including TikTok, Instagram, YouTube, Facebook, and Odnoklassniki (OK.ru).

A user submits a property listing (text and photos) to the bot via Telegram. The system translates the listing into Uzbek, generates a natural-sounding voiceover, assembles a vertical short-form video from the photos, and — only after the user confirms — publishes the result to the connected social media accounts.

Features

  • Automatic translation of property listings into Uzbek
  • AI voiceover generation (ElevenLabs text-to-speech)
  • Vertical short-form video assembly (FFmpeg, 1080×1920)
  • One-click publishing to multiple platforms
  • Operator-controlled access (whitelist of authorized users)

TikTok Integration

This application uses the TikTok Content Posting API with the video.upload and video.publish scopes to publish real estate videos directly to a user-authorized TikTok account. No personal profile data, video lists, or analytics are read from TikTok.

How It Works

  1. The real estate professional sends a property listing and photos to the Sahiy Makler Telegram bot.
  2. The bot translates the listing into Uzbek and produces a voiceover.
  3. FFmpeg combines the photos and voiceover into a 1080×1920 vertical video.
  4. The user previews the result and confirms publishing.
  5. The video is uploaded to the user's authorized TikTok and other connected platforms.
