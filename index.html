How it works

Five steps, fully automated, with one explicit confirmation before anything is published.

1

Submit a listing on Telegram

The agent sends a property listing (text + photos) to the Saxiy Makler Telegram bot. Listings can be in Russian, Uzbek, or English.

2

Translate into Uzbek

The bot calls a large-language-model API to produce a natural-sounding Uzbek caption tailored to TikTok-style audiences.

3

Generate voiceover and render video

ElevenLabs generates the AI voiceover. FFmpeg combines photos, voiceover, captions, and watermark into a 1080×1920 vertical video.

4

Preview and confirm

The bot sends the rendered video back for preview. The agent reviews it and taps Publish — or Edit to tweak the caption or regenerate.

5

Publish to all connected platforms

Once confirmed, the video is uploaded to TikTok (via the Content Posting API) and the other connected platforms — Instagram, YouTube, Facebook, OK.ru — in parallel. Status is reported back in the same Telegram thread.

TikTok integration

Saxiy Makler is a one-way publishing tool. It publishes videos — it does not read profile data, comments, or analytics beyond the open_id that routes a post to the correct account.

Scopes used

Saxiy Makler uses the products required to publish video — TikTok's Content Posting API, which requires Login Kit. The three scopes below are mandatory for these products:

  • user.info.basic — required because the Content Posting API depends on Login Kit. We use only the open_id it returns to route the post to the correct authorized account. No name, avatar, bio, or follower count is read or stored.
  • video.publish — publish a video directly to the user-authorized account (Direct Post)
  • video.upload — included with the Content Posting API product; our flow uses Direct Post

What we do

  • Publish a pre-rendered MP4 via the Content Posting API Direct Post endpoint https://open.tiktokapis.com/v2/post/publish/video/init/
  • Publish as PUBLIC_TO_EVERYONE only after the user taps Publish in the Telegram bot
  • Report the resulting publish_id back to the user

What we do NOT do

  • Read profile fields beyond the open_id (no name, avatar, bio, or follower count)
  • List, search, or download videos from any TikTok account
  • Read comments, likes, shares, or analytics
  • Store TikTok data beyond the configured access token and the open_id
  • Share TikTok-derived data with any third party

Frequently asked questions

Answers to the most common questions about Saxiy Makler.

Is Saxiy Makler free to use?

Saxiy Makler is currently in private beta and access is operated by invitation only. Whitelisted real estate professionals in Uzbekistan can use the service at no charge during the beta period. Pricing details for the public launch will be announced through the Telegram bot.

How long are my listings and videos stored?

Temporary media files (audio, video, images created during processing) are automatically deleted within 24 hours after publishing. Property listing text is not persisted after the video is generated. Full retention details are in the Privacy Policy.

Can I disconnect my TikTok account?

Yes. The operator can revoke the TikTok access token at any time, which immediately stops Saxiy Makler from publishing to TikTok on your behalf. To request a full data deletion, email abdullohsharifov773@gmail.com with the subject "Data Deletion Request".

Which countries does Saxiy Makler operate in?

Saxiy Makler is operated from Tashkent, Uzbekistan, and is built for the Uzbekistan real estate market. The Uzbek-language voiceover and translation are tuned for local terminology and place names.

Will my video include a watermark?

Yes — every video includes a small Saxiy Makler watermark in the corner. This identifies the source of the AI-generated content and is consistent with TikTok's branding-disclosure guidelines.

What happens if TikTok rate-limits my account?

If TikTok returns a 429 (rate limit) or 403 (forbidden) response, Saxiy Makler automatically pauses publishing to that account for 24 hours to avoid triggering shadowban heuristics. The Telegram bot will notify you and resume automatically.

Can I edit the AI-generated caption before publishing?

Yes. The Telegram bot sends a preview with the generated caption. You can tap Edit to modify the text or regenerate it before tapping Publish. Nothing is published until you explicitly confirm.

Does Saxiy Makler read my TikTok analytics or comments?

No. Saxiy Makler is a one-way publishing tool. The user.info.basic scope is requested only because TikTok's Login Kit is mandatory for the Content Posting API — and from it we use only the open_id that tells us which account to post to. We never read your name, avatar, follower count, comments, likes, or analytics.
